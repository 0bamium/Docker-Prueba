# Docker-Prueba

Este trabajo fue realizado entre `Vicente Bravo` y `José Méndez`.


### Descripcion del trabajo

Crear un proyecto Docker con dos servicios (web + base de datos) que funcionen juntos. La web debe
mostrar un logo creado por la pareja y una tabla LOG cargada desde la base de datos. Esta tabla debe
incluir: hora, actividad, estado e imagen (visible en cada fila).
